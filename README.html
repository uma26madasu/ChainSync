ChainSync Universal Coordination Platform API

  • Automated alert systems with coordination scheduling

Prerequisites

  • Anypoint Studio 7.x with Mule Runtime 4.4.0
  • Java 8 or higher
  • Maven 3.6+

Installation

Clone the repository

git clone https://github.com/your-org/chainsync-platform-api.git cd chainsync-platform-api

Import into Anypoint Studio

  • File ‚Üí Import ‚Üí Anypoint Studio ‚Üí Anypoint Studio project from File System
  • Select the project folder

Configure API Keys (Optional - for real-time data)

# src/main/resources/config.properties openweathermap.api.key=YOUR_API_KEY_HERE airvisual.api.key=YOUR_API_KEY_HERE

üîë How to Get API Keys:

üå§Ô∏è OpenWeatherMap (Weather Data)
  • Visit: https://openweathermap.org/api
  • Sign up for free account
  • Verify email
  • Get API key from dashboard
  • Wait 10 minutes for activation
üå¨Ô∏è AirVisual (Air Quality Data)
  • Visit: https://www.iqair.com/air-pollution-data-api
  • Get free API key
  • Fill registration form
  • Verify email
  • Access key in dashboard

  • ‚úÖ App works without keys (uses mock data)
  • ‚úÖ Rate limits: OpenWeatherMap (1K/day), AirVisual (10K/month)

Run the Application

  • Wait for deployment completion

üì° API Endpoints

Environmental Monitoring

Method Endpoint Description
GET /environmental-data Get all monitoring stations data
GET /environmental-data/{stationId} Get specific station data
GET /emergency-alerts Get active environmental alerts
POST /emergency-alerts Create manual environmental alert

Fleet Management

Method Endpoint Description
GET /fleet-monitoring Get all vehicle data
GET /fleet-monitoring/{vehicleId} Get specific vehicle data
GET /fleet-monitoring/enhanced Get vehicles with environmental context
GET /driver-performance Get driver analytics

Fleet Coordination

Method Endpoint Description
GET /fleet-coordination/recommendations Get coordination recommendations
GET /fleet-coordination/driver-safety-alerts Get driver safety alerts
GET /fleet-coordination/route-optimization Get route optimization suggestions

Alerts & Health

Method Endpoint Description
GET /fleet-alerts Get active fleet alerts
POST /fleet-alerts Create fleet alert
GET /health Get platform health status

üíª Usage Examples

Get Environmental Data

curl -X GET "http://localhost:8081/api/environmental-data" \ -H "Accept: application/json"

Get Fleet Recommendations

curl -X GET "http://localhost:8081/api/fleet-coordination/recommendations" \ -H "Accept: application/json"

Create Emergency Alert

curl -X POST "http://localhost:8081/api/fleet-alerts" \ -H "Content-Type: application/json" \ -d '{ "vehicleId": "TRUCK_001", "alertType": "EMERGENCY", "alertLevel": "CRITICAL", "triggerCondition": "Vehicle breakdown" }'
